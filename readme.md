# noctisyn

a terminal UI for scraping web novels and generating EPUB files.

![noctisyn](https://ftp.jabuxas.com/output.gif-1765568237.200434-tpTG2dLv.gif)

## features

- search novels from novelfull.net and novgo.net
- interactive TUI with split-panel layout
- async chapter downloads with job tracking

## installation

```sh
    go install github.com/jabuxas/noctisyn/cmd/noctisyn@latest
```
